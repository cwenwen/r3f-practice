## Get started

Clone the repository:

```sh
git clone https://github.com/cwenwen/r3f-practice.git
cd r3f-practice
```

Install dependencies:

```sh
yarn install
```

Start the development server on port 5173:

```sh
yarn dev
```

Build for production:

```sh
yarn run build
```
